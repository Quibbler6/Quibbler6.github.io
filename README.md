# Minimal Personal Homepage Template

This is a static personal homepage template inspired by the calm academic style of `b1ueber2y.me`: centered narrow content, white background, quiet opacity, large letter-spaced headings, and a thin bordered navigation bar.

## Files

- `index.html` - landing page
- `about_me.html` - short personal bio
- `academic.html` - research profile and project list
- `1923.html` - photography gallery page
- `css/homepage.css` - shared visual system
- `js/site.js` - shared last-update helper
- `assets/` - local placeholder visuals
- `vendor/font-awesome/` - local Font Awesome 4.7 icons

## Customize

Replace `Your Name`, `your.name at example dot com`, social links, project titles, and `yourdomain.me`. Put photography files in `assets/1923/`, then edit the `<figure class="img-box">` entries in `1923.html` to point to your photos and captions. The template has no build step, so opening `index.html` in a browser is enough.
